# 🌍 Projeto Vida Nômade

Repositório dedicado à organização, automação e gestão da minha jornada como nômade pelo Brasil. 

## 🎯 Objetivos
* **Gestão Financeira:** Controle de gastos em viagens e freelas de bartender.
* **Mapeamento de Voluntariados:** Registro de experiências (como a de Floripa) e próximos destinos.
* **Automação:** Scripts em Python para facilitar minha rotina e estudos de tecnologia.

## 🛠️ Tecnologias & Ferramentas
* Google Sheets (Planejamento Inicial)
* Markdown (Documentação)
* Python (Futuras automações)
